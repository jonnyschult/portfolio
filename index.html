Philosphy

Ux Ui

What makes a good user experience? What the user expects often determines how the user feels about a website. As such, user experience is a dynamic process of expectation and experience. My goal in developing websites is to meet and exceed user expectation by creating a website in-line with its purpose and meticulously and empirically monitored.

Code

There are many different ways to achieve the same results on a webpage. But quality programming goes beyond merely getting results once. Good programming should be transparent, responsive to it's content, and deliver functionality effeciently. A lot of this comes down to the structure and logic of the code itself, but even the best laid paths benefit from a map. Rigorous documentation ensures that the code can be comprehended, used, and improved by whoever is maintaining it. As such, I am developing and improving upon documentation methods for my code.

Work

Business environments can be stressful. Often times this stress is compounded by poor working relationships. It is my philosophy that human productivity, and life in general, is best fostered in an enthusiastic, collaborative and open environment. When these things are done well, meaning and quality of life comes along with productivity. As professional, I conduct my work in a way which brings meaning and productivity to myself and my colleagues.
